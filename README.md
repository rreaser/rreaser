## üìà Getting Your Site to Show Up on Google

### Step 1: Upload Everything to GitHub

Make sure all files are uploaded to your `rreaser.github.io` repository, including:
- All HTML files
- All CSS files
- script.js
- **sitemap.xml** (new!)
- **robots.txt** (new!)
- profile.jpg
- cv.pdf

### Step 2: Submit to Google Search Console

1. Go to: https://search.google.com/search-console/
2. Click "Start now" and sign in with a Google account
3. Enter your URL: `https://rreaser.github.io`
4. Choose "URL prefix" method
5. Verify ownership:
   - Google will give you an HTML file to download
   - Upload this file to your GitHub repository
   - Click "Verify"
6. Once verified, go to "Sitemaps" in the left menu
7. Enter `sitemap.xml` and click "Submit"
8. Click "Request Indexing" for your homepage
